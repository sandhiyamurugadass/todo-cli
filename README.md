# 🗒️ CLI To-Do App

A simple command-line To-Do application built with Python. Tasks are saved locally in a JSON file so they persist between sessions.

## Features
- ✅ Add tasks
- 📋 View all tasks
- ✔️ Mark tasks as done
- 🗑️ Delete tasks
- 💾 Auto-saves to `tasks.json`

## How to Run

```bash
python todo.py
```

## Requirements
- Python 3.x (no extra libraries needed!)
